## ✨ Core Features

- **Premium Typography**: Custom Montserrat-based typography suite for a bold, modern professional aesthetic.
- **Interactive 3D Cards**: Smooth, perspective-shifting project cards powered by custom React hooks and CSS transforms.
- **Dynamic Typewriter**: Smooth role-cycling animation between "Arnabh Kushwaha" and "Founder of Hubrave".
- **Custom Cursor Engine**: Interactive cursor system that responds to UI elements with scale and blend-mode transformations.
- **Glassmorphism UI**: High-fidelity frosted glass effects and scanline overlays for a futuristic "Digital Architect" feel.
- **BCA & AI Focused**: Reflects a background in Data Science and Artificial Intelligence.

## 🚀 Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 8](https://vitejs.dev/)
- **Language**: [TypeScript 6](https://www.typescriptlang.org/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🛠️ Development

### Prerequisites
- Node.js (latest LTS recommended)
- npm or yarn

### Installation
```bash
# Clone the repository
git clone https://github.com/ZenSensi/portfolio.git

# Install dependencies
npm install

# Start development server
npm run dev
```

### Building for Production
```bash
npm run build
```
